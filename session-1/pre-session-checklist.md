## Before You Arrive

### Account Setup (Required)

Make sure all of the following are ready before Session 1. If you get stuck, reach out to me.

- [ ] **Node.js** installed (v20+) — run `node -v` in your terminal to check
- [ ] **npm** installed (comes with Node.js) — run `npm -v` to check
- [ ] **VS Code** installed with these extensions: ESLint, Prettier, Tailwind CSS IntelliSense, Prisma
- [ ] **Git** installed — run `git -v` to check
- [ ] **GitHub account** created — you'll push your project here

## What We'll Build in Session 1

By the end of this session, you'll have a working Next.js application with:

- A complete project scaffolded with TypeScript, Tailwind CSS, and shadcn/ui
- A Postgres database connected via Prisma with your first model (Profile)
- Supabase authentication working (login/logout)
- A protected app shell — logged-in users see the CRM sidebar, logged-out users get redirected to login
- Role-based navigation (Admin, Manager, and Agent see different sidebar links)

This is the foundation everything else builds on. Every future session starts from where this one ends.

## What to Expect

- **Duration:** 3 hours (1.5 hrs → 15-min break → 1.5 hrs)
- **Format:** I code, you follow along on your machine. Ask questions as they come up.
- **Pace:** We'll go step by step. If you fall behind, flag it — I'd rather pause than have you lost.

Bring your laptop fully charged (or bring a charger). Have your terminal and VS Code open when we start.
